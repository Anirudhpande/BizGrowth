BizGrowth User Manual Section 1: Getting Started

1. Getting Started — Registration & Activation

1.1 Creating a Client Account (Corporate / MSME)

To access the B2B marketplace, global trade directories, and products catalog, you must register your enterprise node:

  • Navigate to the landing page and click Register (or access /register).
  • Fill in the registration form with your email, full name, secure password (minimum 8 characters).
  • Select Client Node (Corporate / MSME) from the Enterprise Role dropdown.
  • Click Register Account to create your credentials.
💡
TIP: Stating your official business email (such as sales@yourcompany.com) speeds up the organization approval process in the verification queue.

1.2 Creating a Strategic Consultant Profile

If you are an advisor offering strategy, tax planning, custom clearances, ESG audits, or technology advisory services:

  • Go to the signup form at /register.
  • Fill in details and select Strategic Consultant from the role options.
  • After registering, you will be redirected to /consultants/setup to input expertises, hourly billing rates, and booking slots.
⚠️
IMPORTANT: Consultant profiles remain hidden from public searches until availability slots are defined and saved on the configuration page.
BizGrowth User Manual Section 2: Node Interface

2. Dashboard & Enterprise Node Interface

2.1 Workspace Overview

The dashboard (/dashboard) acts as your command room, displaying the following panels:

  • Enterprise Metrics: Active views of profile validation status, listing statistics, and message notifications.
  • Matchmaking Opportunities: Dynamic partner cards calculated by comparing your tags with listing tags in the database.
  • Booked Sessions: List of upcoming consultation calls and business discussions with calendar details.

2.2 Organization Profile Setup

To list items or submit export details, you must create a verified business profile:

  1. Go to My Organizations or access /organizations.
  2. Click Create Organization Profile to open the form (/organizations/new).
  3. Enter company name, select industry group, input location details, website link, and enterprise description.
  4. Upload logo images and registry credentials.
  5. Submit. Profile goes into the admin verification queue.
⚠️
WARNING: Incomplete profile details or blank website domains will flag the organization as unverified, limiting listing privileges.
BizGrowth User Manual Section 3: B2B Marketplace

3. B2B Marketplace & Smart Matchmaking

3.1 Creating Marketplace Listings

The B2B marketplace (/marketplace) is where MSMEs and Corporates collaborate on trading and inventory options:

  • Navigate to the marketplace board and click Post Opportunity (/marketplace/new).
  • Input listing title and select type: Buy, Sell, Partner, or Barter.
  • Enter description details and specify tag arrays (e.g., Basmati, Export, Logistics).
  • Save. Listing goes live on the marketplace grid.

BizGrowth User Manual Section 4: Import-Export

4. Import-Export & Global Trade Portals

BizGrowth facilitates international trade via dedicated state and global directories:

4.1 India to Global Directory

For Indian manufacturers seeking global markets (/india-to-global):

  • Exporters can view active global buy requests and target import regions.
  • Click Post Export Offer to list item supplies, state origins (e.g., Gujarat, Punjab), packaging limits, and shipping details.

4.2 Global to India Directory

For foreign entities seeking suppliers in India (/global-to-india):

  • International users list component requirements and targeted supply cities.
  • Verified Indian entities can bid on requests and initiate supply contracts directly.

BizGrowth User Manual Section 5: Consultant Hub

5. Consultant Hub & Booking System

5.1 Booking a Session with a Consultant

Businesses requiring financial, compliance, or logistics audits can book sessions with strategic advisors:

  • Go to the Advisor Search registry at /consultants.
  • Filter profiles by core expertise (e.g. Taxations, ESG, customs clearances).
  • Click View Profile and select Book Consultation.
  • Select an available slot and click Proceed to Payment to invoke the booking confirmation modal.

5.2 Availability Slots Setup

Consultants configure weekly availability slots to receive bookings:

  1. Go to your Consultant Workspace.
  2. Select Manage Availability.
  3. Add specific day slots (e.g., Mon/Wed 10:00 AM - 12:00 PM).
  4. Input timezone and click Publish Slots.
💡
TIP: Booking status transitions automatically from Pending to Confirmed upon successful mock payment clearance in the overlay.
BizGrowth User Manual Section 6: Products & Hubs

6. Products, Events & Knowledge Hub

6.1 Product Catalog

Enterprise nodes list product availability for potential wholesale buyers (/products):

  • Click Add Product Listing to open the product editor form.
  • Input target wholesale price, stock quantities, and product category.
  • Saved products appear under your organization's card in search directories.

6.2 Conclaves & Events

Networking conclaves are organized to coordinate buyer-seller meets (/events):

  • Browse upcoming events and webinars.
  • Click Register for Event to reserve attendance.
  • Association nodes can request event host permissions to compile new conclaves.

6.3 Knowledge Hub

The Knowledge Hub (/resources) provides access to regulatory resources:

  • Browse PDFs, guidelines, and articles on international trade tariffs, export taxation, and compliance.
  • Vetted consultants and admins can publish new whitepapers directly using the Resource form.
BizGrowth User Manual Section 7: Messaging

7. Communication & Real-time Messaging

The BizGrowth messaging system (/messages) supports secure client-consultant and buyer-seller communications:

7.1 Initiating an Enterprise Conversation

  1. Navigate to any marketplace listing, product detail card, or consultant profile.
  2. Click the Send Message button.
  3. An inbox thread is initialized automatically.
  4. Type details about pricing, custom parameters, or meeting schedules, and send.
⚠️
IMPORTANT: For safety reasons, do not share credentials or registry passwords inside messaging channels. The support team will never request access keys in a message thread.
BizGrowth User Manual Section 8: Administration

8. Platform Administration & Moderation

The Admin Dashboard console (/admin) is accessible only to accounts flagged with the admin credential in the DB:

8.1 Verification Queue

Administrators review pending organizations to grant verification badges:

  • Navigate to the approval registry on the admin dashboard.
  • Review company registry certifications and domain registrations.
  • Click Verify to approve the profile (adds a blue checkmark to company listings).
  • Click Reject and enter clarification notes to request re-submission.

8.2 Moderation Controls

Admins perform active site moderation:

  • Edit or delete marketplace listings flagged as spam or containing restricted inventory.
  • Vetting consultant profile credentials and removing unverified advisors.
  • Deactivating profiles showing fraudulent patterns or violating trade codes.

BizGrowth User Manual Section 10: Glossary & Help

10. Key Concepts, Glossary & Troubleshooting

10.1 Key Concepts & Glossary

  • Client Node: The enterprise account type created for MSMEs, manufacturers, and corporate businesses.
  • Consultant Node: Vetted strategic advisors offering trade, compliance, audit, and tax consulting services.
  • India to Global: Directory listing export trade cards created by Indian firms targeting international markets.
  • Global to India: Directory listing import requirements created by foreign entities sourcing from India.
  • Smart Matchmaking: Algorithm matching listings by scanning and correlating tag lists in real-time.

10.2 Troubleshooting Guide

Encountered Error Probable Cause Suggested Action
Cannot post opportunities Organization is incomplete or unverified. Go to My Organizations, fill out profile, and wait for verification badge.
Consultant profile not in search Availability slots have not been configured. Go to availability settings, add slots, and save hours.
Admin dashboard is blank Signed-in user lacks admin role in the DB. Verify you are logged in using an official admin account (e.g. admin@bizgrowth.com).
